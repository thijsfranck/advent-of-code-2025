## Installation

> [!NOTE]
> This project requires [Python 3.14](https://www.python.org/downloads/) and [UV](https://github.com/astral-sh/uv).

To install the project dependencies, execute the following command:

```bash
uv venv --allow-existing
uv sync
```

## Tests

To run the tests, execute the following command:

```bash
uv run pytest
```
